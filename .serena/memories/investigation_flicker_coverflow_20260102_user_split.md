# 画面フラッシュ（チラつき）追加切り分け結果（ユーザー確認）

## ユーザーのDevTools切り分け結果
- 背景を「画像」→「単色」にするとチラつきが減る
- ガラス表現のブラー（backdrop-filter の blur）を 0 にするとチラつきが減る

## 解釈（原因の当たり）
- チラつきは「重い背景（高解像度画像）× backdrop-filter のブラー」が、カバーフローのスクロール/3D変形中に毎フレーム再合成されて不安定になるのが主因っぽい。
- 背景画像とブラーのどっちも寄与していて、片方を弱めるだけでも改善する可能性が高い。

## 次の最小修正案（提案）
1) デフォルト背景画像を軽量化（縮小/圧縮/WebP、必要なら image-set）して同じ見た目で負荷を下げる
2) blur 強度を少し下げ、透明度/色で見た目をキープ（glass-distortion 自体は ON のまま）
