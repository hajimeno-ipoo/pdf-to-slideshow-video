## 2025-12-17
- バグ報告対応（画像1/2）
  1) キャンバス編集でオブジェクト追加→適用後、スライド編集プレビューが黒くなる
  2) 適用後にキャンバス編集をOFF（通常に戻す）と、キャンバス編集内容がプレビューに反映されない

### 原因
- (1) `slide` 更新時に `stageSize` を 0 にリセットしていたが、キャンバス表示が継続するとサイズ再計測が走らず、スライドRectが 0 になって黒表示になり得た。
- (2) 「キャンバス表示モード」と「編集できるか（編集ON/OFF）」が同じ `isCanvasMode` に結びついていて、OFFにすると表示も通常モードへ戻ってしまい、`layout` や `space:'canvas'` が見えなくなっていた。

### 修正
- `components/SlideInspector.tsx`
  - `hasCanvasLayout` / `showCanvasStage` を導入し、**表示は layout/背景オブジェクトがある限り維持**、ボタンは**編集ON/OFF**だけを切り替えるように分離。
  - `stageSize` を `slide` 更新でリセットしない（黒表示防止）。
  - 編集OFF時はキャンバス上のオーバーレイ操作（ドラッグ/リサイズ/回転）を無効化。
  - cropタブ中は従来どおり通常プレビュー表示（キャンバス表示は出さない）。

### 検証
- `npm test` / `npm run build` 成功。
