## 2025-12-17
- バグ: キャンバス編集ONでも、画像(オーバーレイ)をドラッグしてスライド外へ出してもスライド内にしか表示されない。レイヤーを一度スライド下へ移すと直る。

### 原因
- 新規追加したオーバーレイが `space` 未指定（= slide 扱い）のまま、スライド枠内でクリップ描画される。
- レイヤー並び替え時は「スライドより下」に移動した瞬間に `space:'canvas'` 化＆座標換算していたため、その操作をすると見えるようになる。

### 修正
- `components/SlideInspector.tsx`
  - キャンバス編集中のドラッグ移動（move）で、slide扱いのオーバーレイがスライド範囲をはみ出したら、**その瞬間に `space:'canvas'` に昇格**して座標/サイズ/見た目（fontSize等）をキャンバス基準に換算。
  - 連続で二重換算しないよう `convertedToCanvasDuringDragRef` を追加。

### 検証
- `npm test` / `npm run build` 成功。
